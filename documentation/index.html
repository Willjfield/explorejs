      Planets

      xpl.SolarSystem(Planet,Time)
      This function returns the position of a planet at a specified time. The planet objects can be accessed from the xpl.planets planets array. The time is entered as a Julian Day. The returned position is described in an XYZ coordinate system with the Z-axis as up and the Sun's barrycenter at the origin. For a demonstration of this function, look at Drawing Planets in P5 //Look through each planet in the Solar System //and find its current position for(var p=0;p< xpl.planets.length;p++){ var planet = xpl.planets[p] var position = xpl.SolarSystem(planet,xpl.now) }

      xpl.planets(Planet,Time)
      xpl.planets is an array of Planet Javascript objects. Planet objects contain all of the data needed to derive the position, rotation and obliqueness of each planet in our Solar System given a certain time. //Look through each planet in the Solar System //and store it as a variable called planet for(var p=0;p< xpl.planets.length;p++){ var planet = xpl.planets[p] }

      xpl.planet(name,number,[keplerian elements],radius,oblique,dayLength,yearLength,[color])
      xpl.planet objects contain the information needed for tracking and drawing planets in our Solar System. They contain the name of the planet and its zero-indexed position in an array ordered by distance from the Sun. Next, it has the Keplerian elements or the numbers that can be used to derive its position at a given time. Finally, it contains information useful for drawing it including its radius, obliquity (how tilted it is), the day length, year length, and a rough approximation of its surface color. //Draw planets in p5 function draw(){ xpl.updateTime() for(var p in xpl.planets){ var planet = xpl.planets[p] var position = xpl.SolarSystem(planet,xpl.now) var radius = planet.radius var planetColor = rgbColor fill(planetColor.r,planetColor.g,planetColor.b) push() translate(position[0],position[3]) ellipse(0,0,radius,radius) pop() } }

      Time

      xpl.now
      xpl.now holds the current time as a Julian Date as of when the script is loaded. Julian dates (abbreviated JD or Jday) are the number of days since noon Universal Time on January 1, 4713 BCE (on the Julian calendar). For a detailed explanation of Julian Dates and an online converter tool, look at The US Naval Obsrvatory's Julian Date Converter //Find planet Earth and determine its current position //as of when the script is executed var planet = xpl.planets[2] var position = xpl.SolarSystem(planet,xpl.now) console.log(position)

      xpl.updateTime()
      Since xpl.now is created when the script loads, you will need to update the time if you need a continous measurement. For instance, an animation can track planets and satellites in real time if xpl.updateTime() is called on each frame. //Find planet Earth and determine its current position //in each draw() loop. xpl.updateTime() will update the value //of xpl.now to the time as of when each draw() loop is executed. function draw(){ xpl.updateTime() var planet = xpl.planets[2] var position = xpl.SolarSystem(planet,xpl.now) console.log(position) }

      xpl.jday(year,month,day,hour,minute,second)
